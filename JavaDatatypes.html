Description

Project work set for “Introduction to Programming” unit at Halesowen College.

During this unit I learnt how to use the Java development kit to compile and run basic Java programs.

One of the tasks during this unit was to demonstrate my understanding of more advanced data types like: Stacks, Queues, Trees, Graphs and Sets.

The code I developed was designed to run as a command line application and displayed a relevant use case for each of the data types.
