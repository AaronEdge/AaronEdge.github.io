Description

Project work set for "Database Design and
Development" unit at Halesowen College.

The scenario was that a car rental company had come to me
looking to improve their current paper based rental system by transferring most
of the paperwork to a computer-based system.

My job was to conduct a feasibility study to find out what
potential impact a computer-based system would have and if it was even possible.
I also had to evaluate the existing solutions available to them and justify building
a custom solution.

I then had to create the design documents, user specification
and design specifications.

Then using the documentation, I had to implement the software
solution. For this I used Java and a SQL database.
